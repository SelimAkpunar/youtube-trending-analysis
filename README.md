# youtube-trending-analysis
This project analyzes YouTube Trending Dataset using Python (Pandas, Matplotlib, Seaborn) to discover patterns in views, categories, engagement, and country-based trends.

## Project Goal
The goal of this project is to explore YouTube trending videos and answer questions like:
- Which categories get the most views?
- How does engagement vary by country?
- What factors influence trending videos?

- ## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

- ## Analysis Performed
- Top trending videos by views
- Category-based analysis
- Country-based comparison
- Like / view ratio analysis
- Time-based trends

- ## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Open Jupyter Notebook:
   jupyter notebook

   ## Project Structure
- data/ → dataset files
- notebooks/ → analysis notebooks
- README.md → project description
